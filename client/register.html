Why must I register for AppLaud?

You've already signed in using Open ID, now you just need to pick a user ID so we can create your own private workspace in the cloud. You don't need to remember your AppLaud user ID, only your OpenID name and password (which we don't store or even see - read more about Open ID on the login page).

AppLaud Cloud is free and open source! Read more about licensing on the FAQ page.

What happens after I register?

You will be taken to your AppLaud home page (bookmark it!) and can immediately start creating AppLaud projects. The starting page will have a Getting Started cheat sheet, and an easy-to-find Help tab.
